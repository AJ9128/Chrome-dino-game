# Chrome Dino Game (Python)

## Overview
A Python-based recreation of the Chrome T-Rex offline game, implementing core gameplay mechanics such as jumping, obstacle generation, collision detection, and scoring.

## Features
- Real-time keyboard input handling
- Obstacle generation and increasing difficulty
- Collision detection and game-over logic
- Score tracking system

## Technologies Used
- Python

## What I Learned
- Game loop design and state management
- Event-driven programming in Python
- Collision detection and basic physics simulation
